# 🛒 Customer Retail Data - Machine Learning Model Comparison

## 📌 Project Overview
This project focuses on applying multiple machine learning models on a retail customer dataset and comparing their performance. The goal is to identify the most effective model for predicting customer behavior.

---

## 📊 Dataset Information
- Dataset: Customer Retail Dataset
- Type: Structured (Tabular Data)
- Features: Customer demographics, purchase history, and behavioral attributes

---

## 🎯 Objectives
- Perform data preprocessing and cleaning
- Apply multiple machine learning models
- Evaluate and compare model performance
- Identify the best-performing model

---

## ⚙️ Workflow

### 1. Data Preprocessing
- Handling missing values
- Encoding categorical variables
- Feature scaling

### 2. Model Implementation
The following models were applied:
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)

### 3. Model Evaluation
- Accuracy Score
- Confusion Matrix
- Precision & Recall
- Model comparison

## 📈 Results
